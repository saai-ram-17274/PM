Summary

At 2026-03-24 03:12:44, user corp/m.henderson authenticated successfully from 185.220.101.42 (Romania, Tor exit node) and again from internal IP 10.18.1.81 within 28 minutes — a physical travel distance that is impossible. This was followed by impossible travel confirmed with OAuth token generation, mailbox access and SharePoint file downloads on sensitive directories. Classified as a True Positive — no authorized remote access or travel activity exists. Immediate account suspension, token revocation, and session termination are required.

Insights

Tor Exit Node Confirmed
Source IP 185.220.101.42 is a confirmed Tor exit node (Bucharest, Romania) with a 97 threat score on AbuseIPDB. Zero prior login history from Romania or any Tor relay for this account in the last 90 days.
2
OAuth Token Theft Pattern
Within 87 seconds of initial login, 3 OAuth tokens were issued to unregistered app IDs requesting Mail.Read and Files.ReadWrite.All scopes. This pattern matches known AiTM phishing post-compromise playbook (Microsoft Threat Intel ref: DEV-0867).

Abnormal Data Exfiltration
24 files downloaded from /Corp/Finance/Sensitive and /Legal/Contracts in 3 minutes — download rate of 6.9 files/min is 14× above m.henderson's 90-day baseline of 0.5 files/min. DLP policy was set to Monitor Only.
4
Impossible Travel Confirmed
Two simultaneous active sessions detected: one from Romania and one from internal IP 10.18.1.81 (New York office), 8,073 km apart within 28 minutes — physically impossible. The internal session is the legitimate user; the external session is the attacker.
